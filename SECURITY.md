# Политика за сигурност (Security Policy)

## Поддържани версии

В момента се поддържат следните версии с фиксове за сигурност:

| Версия | Поддържана |
| :--- | :--- |
| 0.2.x | ✅ |
| 8.11.x | ✅ |
| < 8.11 | ❌ |

## Докладване на уязвимост

Ако откриете проблем със сигурността, моля, НЕ отваряйте public issue. Изпратете доклад на `security@icap-platform.io`.

Ще се опитаме да отговорим в рамките на 48 часа и да предложим решение възможно най-скоро.

# Security Policy

## Supported Versions

The following versions are currently supported with security fixes:

| Version | Supported |
| :--- | :--- |
| 0.2.x | ✅ |
| 8.11.x | ✅ |
| < 8.11 | ❌ |

## Reporting a Vulnerability

If you find a security issue, please DO NOT open a public issue. Send a report to `security@icap-platform.io`.

We will try to respond within 48 hours and provide a fix as soon as possible.
