# Digital Twin Module — ICAP Documentation

## 🏢 Общ Преглед
Модулът **Digital Twin (Дигитален двойник)** в ICAP v0.2.8 позволява виртуална симулация на производствените процеси. Той служи като мост между реалните сензорни данни (IoT) и прогнозния анализ, позволявайки на операторите да тестват сценарии тип "What-if" без риск за реалната продукция.

## 🛠 Функционалност
- **Color Shift Simulation**: Предсказва как промените в температурата, влажността и налягането ще повлияят на Delta E.
- **Risk Assessment**: Автоматично класифицира сценариите като ниско или високо рискови.
- **Recommendations**: Дава проактивни съвети за настройка на машините на база симулираните отклонения.

## 🔬 Симулационен модел
Дигиталният двойник използва комбинация от:
1. **Физични константи**: Влияние на околната среда върху полимеризацията и наситеността.
2. **Машинни профили**: Оптимални работни точки за всяка машина (напр. Machine_P3).
3. **Емпирични данни**: Коефициенти, извлечени от исторически измервания в базата данни.

## 🚀 Употреба през API
Симулациите се достъпват чрез `DigitalTwinService` в ядрото на системата.

```python
# Пример за What-if сценарий
scenario = {
    "name": "High Humidity Test",
    "env": {"humidity": 75.0, "temperature": 24.5},
    "machine": {"pressure": 4.2}
}
results = await digital_twin.run_what_if_analysis(current_state, [scenario])
```

## 📈 Ползи за бизнеса
- **Намаляване на брака**: Ранно откриване на потенциални проблеми при смяна на климатичните условия.
- **Оптимизация на ресурсите**: Намиране на най-енергийно ефективните работни параметри.
- **ISO 9001 Поддръжка**: Документиране на симулациите като част от процеса по качествен контрол.
